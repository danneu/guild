Formatting Cheatsheet

You can **embolden** and *italicize* text.

You can created [hyperlinks](http://example.com)

You can auto-linkify a url 

You can embed images: ![](/img/spinner.gif)

You can create `inline code blocks`, or you can create multiline code blocks by prepending 4 spaces before each line:

    Meet my elephant:

             ___     _,.--.,_
          .-~   ~--"~-.   ._ "-.
         /      ./_    Y    "-. \
        Y       :~     !         Y
        lq p    |     /         .|
     _   \. .-, l    /          |j
    ()\___) |/   \_/";          !
     \._____.-~\  .  ~\.      ./
                Y_ Y_. "vr"~  T
                (  (    |L    j
                [nn[nn..][nn..]
            ~~~~~~~~~~~~~~~~~~~~~~~

You can create unordered lists:

- Item A
- Item B
- Item C

And ordered lists

1. Item A
2. Item B
3. Item C

You can quote somebody:

> I have two questions:
>
> 1. What color is the sky?
> 2. What's the #1 fundamental rule of the guild?

1. Blue
2. Be fonz-cool

You can also organize your post with six levels of headers:

# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

You can create horizontal separators with 3 or more hyphens in a row:

---

Finally, you can click "View Raw" on any post to see the original text.
